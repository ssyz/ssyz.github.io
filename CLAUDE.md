# CLAUDE.md

## Project Overview
Company website for OurDojo (ourdojo.org). Pure HTML/CSS/JS — no frameworks, no build tools, no npm packages. Hosted on GitHub Pages, deploys on push to `master`.

## Rules
- Always ensure implementation is cross-platform compatible — looks good on both web and mobile
- Always validate that the design system of the entire website is cohesive
- Keep zero-dependency philosophy — no npm packages, no build steps, no CSS frameworks
- Use existing CSS custom properties for colors and spacing — never hardcode values
- Use semantic HTML5 elements and follow existing CSS naming conventions
- GA4 analytics script must be included on every new page
- Inline SVGs for icons — no icon libraries
- All external links use `target="_blank" rel="noopener noreferrer"`
- Keep JavaScript minimal and vanilla
